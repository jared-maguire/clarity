# Chrome Web Store Listing

## Name
Clarity — Reader Mode

## Short Description (132 char max)
Turn any page into a clean, dark reader view. Enable once per tab — it stays on for every page you visit.

## Detailed Description

Clarity gives you a distraction-free reading experience on any website. Click the icon, hit Enable, and every page you navigate to in that tab is automatically converted into a clean, comfortable reader layout.

FEATURES

- Persistent per-tab reader mode — enable once, browse freely, every page gets cleaned up automatically
- Dark theme — easy on the eyes with white text on a dark background
- Smart content extraction — finds the article on any page using text density analysis and semantic scoring
- Contrast fixer — preserves original formatting but automatically detects and corrects unreadable color combinations
- Font picker — choose between Sans, Serif, and Mono fonts
- Zero data collection — everything runs locally in your browser, nothing is ever sent anywhere

HOW IT WORKS

1. Click the Clarity icon in your toolbar
2. Click Enable
3. The page converts to a clean reader view
4. Click any link — the next page gets reader mode too
5. Stays active until you disable it or close the tab

Clarity strips away ads, popups, sidebars, and navigation clutter, leaving just the content you came to read. It uses intelligent scoring to find the main article on each page, preserves images and formatting, and ensures everything is readable against the dark background.

No account needed. No configuration required. No data collected. Just clean reading.

## Permission Justifications

### Host permissions (<all_urls>)
Clarity needs access to web pages in order to extract article content and replace the page with a clean reader view. The extension only activates on pages where the user explicitly enables reader mode. No page data is collected or transmitted.

### Scripting
Used to inject the content extraction and reader rendering script into tabs where the user has enabled reader mode.

### Storage
Used to save the user's font preference (Sans/Serif/Mono) locally in the browser.

## Single Purpose Description
Clarity provides a persistent reader mode that converts web pages into a clean, dark, distraction-free reading layout.
