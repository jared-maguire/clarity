# Privacy Policy — Clarity

**Last updated:** March 28, 2026

## Overview

Clarity is a browser extension that converts web pages into a clean reader view. It operates entirely within your browser and does not collect, transmit, or store any personal data.

## Data Collection

Clarity does **not** collect any data. Specifically:

- No browsing history is recorded or transmitted
- No page content is sent to any server
- No personal information is collected
- No analytics or tracking of any kind
- No cookies are set by the extension
- No third-party services are used

## Data Storage

Clarity stores the following data locally in your browser using Chrome's built-in storage APIs:

- **Font preference** (Sans, Serif, or Mono) — stored locally to remember your choice
- **Active reader tabs** — stored in session memory so reader mode persists during your browsing session; cleared when the browser closes

This data never leaves your browser.

## Permissions

- **Host permissions (`<all_urls>`)** — required to inject the reader mode script into any web page you choose to activate it on. No data from these pages is collected or transmitted.
- **Scripting** — used to inject the content extraction and reader rendering script into the active tab.
- **Storage** — used to save your font preference locally.

## Changes

If this policy changes, the update will be posted here with a new date.
